Operating System Scheduling Simulator

Operating System Scheduling Simulator (Java). A Java-based simulation of core operating system components that models process creation, memory loading, and CPU scheduling. The system reads process data from a text file and represents each process using a Process Control Block (PCB). Multiple scheduling algorithms are implemented, including First Come First Serve (FCFS), Round Robin (RR), and Priority Scheduling (PS). The project uses multithreading, synchronized queues, and file I/O to simulate realistic OS behavior.
